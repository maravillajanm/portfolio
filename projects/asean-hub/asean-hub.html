H&PS ASEAN People & Learning Hub

Introduction

H&PS ASEAN People & Learning Hub is an Enterprise Website that is created from scratch using HTML, CSS and JavaScript.

I worked with a fellow developer in designing and creating three webpages (Home, Career Counseling Journey, and Career Mentors).

Approach

The approach we did first was to research for professional looking web designs considering we are building webpages for enterprise use. We created it from scratch using HTML, CSS and JavaScript and host it in a Sharepoint site. Subsequently, we've added some frameworks to achieve better experience in navigating the site and its components

Full list of technologies used: HTML, CSS, JavaScript, Bootstrap, JQuery, FlexSlider, and FontAwesome

Conclusion

Created three webpages. Censored names and pictures due to privacy reasons for enterprise websites.
